# GA-Optimized Neural Network for Credit Scoring

This project uses a **Genetic Algorithm (GA)** to evolve the architecture of a **Multi-Layer Perceptron (MLP)** for credit scoring. The study first replicates prior research on the **Australian and German credit datasets**, comparing **logistic regression, a default MLP, and a GA-optimized MLP** using **ROC/AUC** as the evaluation metric. After reproducing the published setup, the same GA-based method is applied to additional credit datasets to examine whether automatically discovered architectures generalize across datasets.
