# GA-Optimized Neural Network for Credit Scoring

This project uses a **Genetic Algorithm (GA)** to automatically evolve the architecture of a **Multi-Layer Perceptron (MLP)** for credit scoring. The approach replicates prior research conducted on the **Australian and German credit datasets**, comparing **logistic regression, a default MLP, and a GA-optimized MLP** using **ROC/AUC** as the evaluation metric. After reproducing the published setup, the same GA-based method is applied to additional credit datasets to examine whether automatically discovered architectures generalize across datasets.

This is a research project used for a course named Introduction to Generative AI.
